# Cline Directory Map

At the top level of the `src/coding-agents/cline` submodule:

- `src/` - extension host code (controller, task loop, tools, prompts, storage).
- `webview-ui/` - React webview UI bundled into the extension.
- `cli/` - Go CLI and packaging for standalone usage.
- `proto/` - protocol definitions and generated artifacts.
- `tests/` - VS Code extension tests and fixtures.
- `testing-platform/` - test harness utilities and orchestration scripts.
- `evals/` - evaluation harnesses and test inputs.
- `assets/` - icons and media assets.
- `locales/` - localized README files.
- `scripts/` - build and helper scripts.
- `docs/` - product docs and walkthroughs.
- `standalone/` - standalone build artifacts and packaging helpers.
